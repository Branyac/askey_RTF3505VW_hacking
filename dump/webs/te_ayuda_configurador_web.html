Ayuda configurador web

¿Tienes dudas sobre las opciones que te mostramos para hacer las configuraciones? Desde aquí te ayudamos a que entiendas qué son y para qué sirven:

  • WiFi

      • Mediante esta opción podrás configurar todos los bloques de tu red inalámbrica

      Identificador de Red inalámbrica (nombre WiFi):

      Nombre que recibe tu red inalámbrica. Cualquier equipo inalámbrico que quiera conectarse a una red inalámbrica debe conocer el nombre WiFi de la red. Es un código incluido en todos los paquetes de una red inalámbrica para identificarlos como parte de esa red. Mediante el nombre WiFi se pueden distinguir las distintas redes inalámbricas existentes. Por defecto viene definido como: "MOVISTAR_XXXX".

      Ocultar nombre WiFi:

      Mediante esta opción podrás ocultar el nombre WiFi para que no sea visible por nadie y evitar que alguien pueda localizar tu red inalámbrica y conectarse a ella.

      Clave WiFi alfanumérica:

      Combinación de caracteres alfanuméricos que te permite acceder a la red inalámbrica. Además se incluye una opción que permite visualizar el nivel de seguridad de la contraseña que has elegido de acceso a tu red inalámbrica. Se indican 3 niveles de seguridad, Bajo, Medio y Alto. Te recomendamos siempre que sea posible que el nivel de seguridad de tu contraseña WiFi sea Alto.

      Tipo de cifrado:

      o WPA2: sistema de cifrado más moderno para proteger las redes inalámbricas. Se emplea para clientes inalámbricos 802.11n. Con WPA2 puedes utilizar tanto el algoritmo de encriptación AES como TKIP.

      o WPA: sistema de cifrado para la protección de redes inalámbricas que surgió para corregir las deficiencias del cifrado WEP. Se emplea para clientes inalámbricos 802.11g. Con WPA puedes emplear TKIP.

      o WEP: primer sistema de cifrado incluido en el estándar IEEE 802.11. Actualmente ofrece una protección demasiado débil y relativamente fácil de vulnerar. Este tipo de cifrado funciona con clientes 802.11b y 802.11g. Se recomienda no usar WEP para proteger tu red inalámbrica si tienes alternativa, ya que su protección es demasiado débil y fácil de vulnerar.

      o Sin cifrado: tu red inalámbrica no tiene ningún tipo de cifrado. Te recomendamos no utilizar esta opción.

      Tipo de encriptación

      o AES (Advanced Encription Standard): es un algoritmo de cifrado por bloques, donde el tamaño de la clave y del bloque son configurables. Se basa en varias operaciones realizadas en bloques de datos de 16 bytes. Actualmente es más seguro que TKIP.

      o TKIP(Temporal Key Integrity Protocol): este algoritmo fue especialmente diseñado para evitar los ataques se producían contra WEP. TKIP empieza con una clave temporal de 128 bits que comparte entre los clientes y los puntos de acceso. Combina esta clave temporal con la dirección MAC de cada cliente y añade a su vez un vector de inicialización para producir la clave final con la que se van a cifrar los datos.

      o TKIP+AES: combinación de los dos algoritmos explicados anteriormente.

      Número canal WiFi:

      Por defecto, el canal de radio está en modo automático (seleccionará de manera automática el canal WiFi disponible menos cargado). También podrás seleccionar de manera manual el canal en el cual quieres que opere tu red inalámbrica.

      Código QR:

      Escaneando el código QR mediante una aplicación lectora de códigos QR que tengas instalada en tu equipo inalámbrico, podrás registrar los datos de la red inalámbrica como nombre WiFi, clave inalámbrica y tipo de cifrado para poder compartirlo con aquellas personas que quieras que puedan conectarse a tu WiFi. Además si tu aplicación de escaneo lo permite, podrás conectarte automáticamente a la red inalámbrica con solo escanear este código.

      WPS (WiFi Protected Setup):

      Mecanismo de intercambio seguro de configuración inalámbrica. Este método permite facilitarte la asociación de un cliente inalámbrico (equipo que conecte con el router y sea compatible con WPS) sin necesidad de introducir la clave inalámbrica en dicho cliente o tener que emplear complicados procesos de configuración.

      Para saber cómo utilizar este método consulta la Guía de instalación que te ha sido entregada junto al router.

      Filtrado MAC:

      Mediante esta opción puedes restringir los equipos que pueden acceder a tu red inalámbrica. Para ello se crea una tabla con las direcciones MAC de los dispositivos que pueden acceder a la red. Para poder utilizar esta opción de manera correcta realiza los siguientes pasos:

      1º Escribe la dirección MAC del equipo al cual quieres permitir el acceso en tu red inalámbrica.

      2º Pulsa el botón "Añadir". Una vez pulsado se debe crear una entrada en la tabla con esa dirección MAC.

      3º Para eliminar una entrada de la tabla de manera definitiva pulsa el botón "Borrar". También tienes la opción de eliminar todas las entradas de la tabla de manera rápida mediante el botón "Borrar todos".

  • Puertos

      • Mediante esta opción podrás abrir los puertos que desees para equipos conectados a tu red.

      Nombre de la regla de puertos: introduce el nombre que quieras asignarle a la regla de puertos y te permita identificarla de manera intuitiva. Cada una de estas reglas ocupará una entrada en la tabla.

      Protocolo: selecciona el protocolo al que se aplica la regla: TCP, UDP o ambos.

      Abrir el puerto o rango de puertos (externo) (WAN): indicar si la regla está definida por un puerto o por un rango de estos. Para abrir un rango de puertos usar el formato 5001: 5010.

      Abrir el puerto o rango de puertos (interno)(LAN): indicar si la regla está definida por un puerto o por un rango de estos. Para abrir un rango de puertos usar el formato 5001: 5010.

      Dirección IP o Rango direcciones IP local del dispositivo(s): Indicar la dirección IP del equipo asociados al router para el cual se quiere aplicar la regla de puertos definida.

      Pulsa el botón "Añadir". Una vez pulsado se debe crear una entrada en la tabla con la regla de puertos definida.

      Si deseas deshabilitar temporalmente la regla de puertos, pulsa el botón "Desactivar". Cuando quieras volver a habilitarla pulsa el botón "Activar".

      Para eliminar una entrada de la tabla de manera definitiva pulsa el botón "Borrar". También tienes la opción de eliminar todas las entradas de la tabla de manera rápida mediante el botón "Borrar todos"

  • Red Local

      • Mapa de red local:

      Opción que permite mostrar de manera gráfica e intuitiva todos los dispositivos conectados tu red local. Te permitirá detectar elementos desconocidos o intrusos que se hayan conectado a tu Router.

      Configuración red local:

      Mediante este apartado podrás configurar tu LAN, cambiar la IP local (LAN) y su máscara de subred. Mediante este apartado también podrás configurar el servidor DHCP. Aquí vienen pre-configuradas la dirección inicial y final de rango de direcciones que el router otorgará a los equipos que se conecten a él. También permite configurar los servidores DNS pero recomendamos no modificar los que vienen configurados por defecto

      Dirección IP Gateway:

      Es la dirección IP privada del router, es decir la dirección que toma el router dentro de la LAN. Por defecto tomará el valor 192.168.1.1.

      Máscara de subred:

      Por defecto su valor es 255.255.255.0 lo que equivale a 253 direcciones IP posibles dentro de nuestra LAN. Para cambiar el número podemos reducir el nº de ips disponibles en nuestra LAN modificando la máscara de subred.

      DHCP:

      Se configurará el estado del servidor DHCP del router (ACTIVADO/DESACTIVADO). Por defecto está activado. En este caso es necesario configurar las direcciones inicial y final del rango de direcciones que el router otorgará a los equipos que se conecten a él.

      Servidores DNS:

      Estos servidores permiten traducir de un nombre de dominio a una dirección IP.

  • Multipuesto/Monopuesto

      • Mediante esta opción podrás configurar tu línea en "Multipuesto" o bien en "Monopuesto" seleccionando la opción deseada.

      Multipuesto:

      Mediante esta configuración podrás crear una red de varios ordenadores en tu LAN. Es la configuración por defecto de tu Router. En esta configuración, en principio, todos los puertos están "cerrados" (salvo aquellos que siempre están abiertos para permitir la navegación por ejemplo la navegación por Internet y recibir correo electrónico). La IP pública no se la queda el ordenador con el que vas a navegar, se la guarda el propio Router. Este a su vez tiene una IP privada (192.168.1.1), de esta manera cada ordenador o equipo para navegación debemos configurarlo para que utilice como puerta de enlace la dirección IP del Gateway 192.168.1.1 Este modo de configuración es más seguro que el método monopuesto.

      Monopuesto:

      Mediante el uso de este modo podrás tener tu ordenar o equipo cliente conectado directamente a Internet. Este modo permite que tu equipo adquiera directamente la IP pública evitando utilizar NAT (Network Address Translation) para navegar por Internet. Este método sólo permite tener un equipo conectado al Router, consiguiendo una navegación totalmente transparente. Esto es útil si sólo dispones de un equipo en tu red y quieres utilizar programas avanzados que requieran configuración de puertos. Debes configurar una conexión de red en su PC del tipo Banda Ancha (PPPoE), el nombre de usuario y contraseña para la conexión PPPoE provista por Movistar.

      Este método es menos seguro que el Multipuesto.

  • IPv6

      • IPv6 es la nueva versión del protocolo de internet, actualmente la versión de este protocolo es la v4. Debido al gran crecimiento de usuarios de Internet y dispositivos conectados a la red, implica que el rango de direcciones de IPv4 se termine agotando. IPv4 dispone de 2^32 direcciones (direcciones con una longitud de 32 bits, es decir, 4.294.967.296 direcciones) en cambio IPv6, cuyas direcciones son de longitud de 128 bits dispone de un total de 2^128 posibles direcciones.

      El despliegue de IPv6 se irá realizando gradualmente con un periodo de convivencia con IPv4. Ipv6 se irá asentando a medida los equipos de cliente y servicios se vayan adaptando a esta nueva versión del protocolo IP.

      Red local IPv6

      La dirección IPv6 está formada por un total de 128 bits, dividida en dos partes de 64 bits cada una. Los 64 bits de mayor peso los constituyen los 48 bits que entrega el proveedor de acceso o administrador de la red. Los siguientes 16 bits indican la subred concreta. Los 64 bits de menor peso se configuran automáticamente por medio del identificador de interfaz.

      o Dirección Local IPv6:dirección en formato IPv6 que se mostrará cuando no tengamos acceso a Internet.

      o Dirección global IPv6:dirección que se mostrará cuando tengamos acceso a Internet con dirección IPv6. Este concepto de direcciones globales en IPV6 es lo mismo al de direcciones públicas en IPV4.

      o Longitud prefijo de subred: valor decimal que indica cuantos bits contiguos de la parte izquierda de la dirección IPv6 componen el prefijo. Dejando el valor 64 se indica que únicamente tenemos una subred

      o Autoconfiguración(stateless):Este modo no requiere ninguna configuración manual de hosts ni precisa servidores adicionales. Permite a un host crear su propia dirección mediante combinación de información disponible de manera local o información publicada por los routers. Es decir permite al equipo configurarse por sí mismo, pudiendo establecer comunicación con el resto de dispositivos de la misma red

      o Rango fijo (stateful):método que dispone de un servidor DHCPv6 donde podrá establecer el rango de direcciones que están permitidas y serán asignadas dentro de la red local LAN). Es necesario establecer la dirección de inicio y fin de rango en los campos correspondientes

      Puertos IPv6

      Mediante esta opción podrás abrir los puertos que desees para equipos conectados a tu red en la configuración IPv6.

      o Nombre de la regla de puertos:introduce el nombre que quieras asignarle a la regla de puertos y te permita identificarla de manera intuitiva. Cada una de estas reglas ocupará una entrada en la tabla.

      o Dirección IPv6 global del dispositivo:Indicar la dirección IP v6 global del dispositivo al cual se quiere aplicar la regla de puertos definida.

      o Protocolo: selecciona el protocolo al que se aplica la regla: TCP, UDP o ambos

      o Abrir el puerto o rango de puertos:indicar si la regla está definida por un puerto o por un rango de estos. Para abrir un rango de puertos usar el formato 5001: 5010

      Pulsa el botón "Añadir". Una vez pulsado se debe crear una entrada en la tabla con la regla de puertos definida

      Si deseas deshabilitar temporalmente la regla de puertos, pulsa el botón "Desactivar". Cuando quieras volver a habilitarla pulsa el botón "Activar".

      Para eliminar una entrada de la tabla de manera definitiva pulsa el botón "Borrar". También tienes la opción de eliminar todas las entradas de la tabla de manera rápida mediante el botón "Borrar todos"

  • Cambio contraseña Router
      • Cambio contraseña Router:

      Mediante esta opción podrás cambiar la contraseña de acceso a tu router. Por defecto la contraseña de acceso, es la definida en la pegatina que podrás encontrar debajo del router denominada "Datos de acceso al router". La nueva contraseña que establezcas para entrar a tu Router debe ser alfanumérica y se recomienda que sea de al menos 8 caracteres.

  • Actualizaciones Firmware

      • Esta opción permite que actualizar la versión Firmware de tu Router de dos formas diferentes:

      Actualiza el Firmware desde tu PC:

      Si dispones de la última versión Firmware y deseas actualizar tu Router de manera manual, pulsa el botón "Buscar" para seleccionar el archivo y cargarlo en tu Router.

      Actualiza el Firmware desde la red de Movistar:

      Mediante esta opción, si hay nuevas versiones Firmware con las que puedes actualizar tu Router, se te indicará mediante un mensaje indicando la última versión de FW disponible. Únicamente debes pulsar el botón "Actualizar" y tu equipo quedará actualizado a la última versión disponible en ese momento. Si al pulsar la opción "Actualiza el Firmware desde la red de Movistar" no se detecta ninguna actualización, se te informará de esto y será suficiente con que pulses el botón "Cerrar".

  • Otras funcionalidades
      • Crear perfil o recuperar uno guardado

      Esta opción te permitirá guardar en un archivo.img la configuración concreta del router, con todos los parámetros para recuperarla más adelante. Un perfil es un conjunto de reglas y parámetros que se aplican para crear una configuración determinada. Se guardan todos los parámetros que definen tu router, tales como los de la red inalámbrica, configuración de red local, filtrado MAC, apertura de puertos, etc.

      Puedes guardar este perfil con el nombre que desees y cargar el perfil cuando lo desees presionando el botón "Guardar". Al hacer click en este botón se te mostrará un resumen del perfil donde debes añadir el nombre que quieras darle al perfil. Pulsa el botón "Aceptar" para almacenarlo. Si finalmente no quieres almacenarlo haz click en el botón "Cancelar".

      Para recuperar un perfil previamente guardado haz click en el botón "Cargar" y selecciona uno de los perfiles que se muestren. Cuando selecciones uno de ellos se te mostrará un resumen con todos los parámetros. Si deseas guardar el perfil haz click en "Aceptar" y si finalmente no deseas guardarlo haz click en "Cancelar"

      Restaurar a valores a fábrica

      Esta opción te permite restaurar la configuración por defecto de fábrica del Router. Debes tener en cuenta que si empleas está opción, se borrarán todos los parámetros configurados en tu Router, como por ejemplo la contraseña de acceso al Router, nombre y contraseña de tu red inalámbrica, etc.tomando de nuevo los valores de la configuración inicial.

      Firewall

      Podrás configurar el Firewall que trae el router para proteger nuestro PC de ataques externos. Te recomendamos no desactivarlo para tener protegido tu PC. Si deseas configurarlo solo tienes que hacer click aquí, o bien a través de la opción: Otras funcionalidades-->Firewall

      Protocolo UPnP

      Esta opción te permitirá activar/desactivar la funcionalidad conocida como Universal Plug and Play (UPnP).

      Permite descubrir de manera automática un dispositivo nuevo conectado en tu red local y puedan ser utilizados casi de manera instantánea sin ser configurados. Una vez que el nuevo equipo se haya configurado correctamente, puedes desactivarlo. Para más información consulta el manual del fabricante del router.

      • Ayuda configurador web:

      Tutorial sobre cada uno de los apartados del configurador web.

      Acerca de:

      Resumen con información relevante acerca de la configuración de tu Router y tu conexión.

      Este apartado muestra los siguientes parámetros:

      Información del equipo:

      Permite identificar tu router y las diferentes versiones de Hardware, Software y Firmware, etc. También podrás identificar de una manera rápida si tu equipo está conectado o no a Internet.

      Información WAN:

      Dirección IP WAN (Internet):Dirección IP pública de tu router.

      Información Red Local (LAN):

      Máscara de subred: muestra el valor de tu máscara de subred.

      DHCP: en caso de estar activado el servidor DHCP mostrará las direcciones de inicio y final de rango de direcciones que se pueden asignar a los equipos asociados a tu Router.

      Servidor DNS:muestra el valor de tu servidor DNS.

      Para más información sobre estos parámetros de tu red local consulta el apartado "Red Local" de este tutorial.

      Información Red inalámbrica

      Estado WiFi: muestra si estás o no conectado de manera inalámbrica.

      Nombre WiFi: Nombre de tu red inalámbrica.

      Oculto Nombre WiFi:Te indica si tienes o no oculto el nombre que identifica tu red inalámbrica.

      Tipo cifrado: muestra el cifrado que has establecido a tu red inalámbrica.

      Encriptación: muestra la encriptación establecida a tu red inalámbrica

      Para más información sobre estos parámetros de tu red inalámbrica consulta el apartado "WiFi" de este tutorial

      Información Movistar TV

      Servicio contratado:muestra si el cliente tiene un servicio Trío o un Dúo contratado

      Tipo de direccionamiento:Indica el tipo de direccionamiento de la línea del cliente. El direccionamiento dinámico es el más habitual. Para tener direccionamiento estático, el cual deberás contratar con Movistar y configurar los valores que te han proporcionado.

      Servicio de voz:Indica si tienes servicio de Voz sobre IP

      Número de Teléfono:muestra tu número de teléfono

      Información de acceso

      Id_ONT: número de acceso a tu ONT.

      • Mapa de red local:

      Opción que permite mostrar de manera gráfica e intuitiva todos los dispositivos conectados tu red local. Te permitirá detectar elementos desconocidos o intrusos que se hayan conectado a tu Router.

      Configuración red local:

      Mediante este apartado podrás configurar tu LAN, cambiar la IP local (LAN) y su máscara de subred. Mediante este apartado también podrás configurar el servidor DHCP. Aquí vienen pre-configuradas la dirección inicial y final de rango de direcciones que el router otorgará a los equipos que se conecten a él. También permite configurar los servidores DNS pero recomendamos no modificar los que vienen configurados por defecto

      Dirección IP Gateway:

      Es la dirección IP privada del router, es decir la dirección que toma el router dentro de la LAN. Por defecto tomará el valor 192.168.1.1.

      Máscara de subred:

      Por defecto su valor es 255.255.255.0 lo que equivale a 253 direcciones IP posibles dentro de nuestra LAN. Para cambiar el número podemos reducir el nº de ips disponibles en nuestra LAN modificando la máscara de subred.

      DHCP:

      Se configurará el estado del servidor DHCP del router (ACTIVADO/DESACTIVADO). Por defecto está activado. En este caso es necesario configurar las direcciones inicial y final del rango de direcciones que el router otorgará a los equipos que se conecten a él.

      Servidores DNS:

      Estos servidores permiten traducir de un nombre de dominio a una dirección IP.

  • Multipuesto/Monopuesto

      • Mediante esta opción podrás configurar tu línea en "Multipuesto" o bien en "Monopuesto" seleccionando la opción deseada.

      Multipuesto:

      Mediante esta configuración podrás crear una red de varios ordenadores en tu LAN. Es la configuración por defecto de tu Router. En esta configuración, en principio, todos los puertos están "cerrados" (salvo aquellos que siempre están abiertos para permitir la navegación por ejemplo la navegación por Internet y recibir correo electrónico). La IP pública no se la queda el ordenador con el que vas a navegar, se la guarda el propio Router. Este a su vez tiene una IP privada (192.168.1.1), de esta manera cada ordenador o equipo para navegación debemos configurarlo para que utilice como puerta de enlace la dirección IP del Gateway 192.168.1.1 Este modo de configuración es más seguro que el método monopuesto.

      Monopuesto:

      Mediante el uso de este modo podrás tener tu ordenar o equipo cliente conectado directamente a Internet. Este modo permite que tu equipo adquiera directamente la IP pública evitando utilizar NAT (Network Address Translation) para navegar por Internet. Este método sólo permite tener un equipo conectado al Router, consiguiendo una navegación totalmente transparente. Esto es útil si sólo dispones de un equipo en tu red y quieres utilizar programas avanzados que requieran configuración de puertos. Debes configurar una conexión de red en su PC del tipo Banda Ancha (PPPoE), el nombre de usuario y contraseña para la conexión PPPoE provista por Movistar.

      Este método es menos seguro que el Multipuesto.

  • IPv6

      • IPv6 es la nueva versión del protocolo de internet, actualmente la versión de este protocolo es la v4. Debido al gran crecimiento de usuarios de Internet y dispositivos conectados a la red, implica que el rango de direcciones de IPv4 se termine agotando. IPv4 dispone de 2^32 direcciones (direcciones con una longitud de 32 bits, es decir, 4.294.967.296 direcciones) en cambio IPv6, cuyas direcciones son de longitud de 128 bits dispone de un total de 2^128 posibles direcciones.

      El despliegue de IPv6 se irá realizando gradualmente con un periodo de convivencia con IPv4. Ipv6 se irá asentando a medida los equipos de cliente y servicios se vayan adaptando a esta nueva versión del protocolo IP.

      Red local IPv6

      La dirección IPv6 está formada por un total de 128 bits, dividida en dos partes de 64 bits cada una. Los 64 bits de mayor peso los constituyen los 48 bits que entrega el proveedor de acceso o administrador de la red. Los siguientes 16 bits indican la subred concreta. Los 64 bits de menor peso se configuran automáticamente por medio del identificador de interfaz.

      o Dirección Local IPv6:dirección en formato IPv6 que se mostrará cuando no tengamos acceso a Internet.

      o Dirección global IPv6:dirección que se mostrará cuando tengamos acceso a Internet con dirección IPv6. Este concepto de direcciones globales en IPV6 es lo mismo al de direcciones públicas en IPV4.

      o Longitud prefijo de subred: valor decimal que indica cuantos bits contiguos de la parte izquierda de la dirección IPv6 componen el prefijo. Dejando el valor 64 se indica que únicamente tenemos una subred

      o Autoconfiguración(stateless):Este modo no requiere ninguna configuración manual de hosts ni precisa servidores adicionales. Permite a un host crear su propia dirección mediante combinación de información disponible de manera local o información publicada por los routers. Es decir permite al equipo configurarse por sí mismo, pudiendo establecer comunicación con el resto de dispositivos de la misma red

      o Rango fijo (stateful):método que dispone de un servidor DHCPv6 donde podrá establecer el rango de direcciones que están permitidas y serán asignadas dentro de la red local LAN). Es necesario establecer la dirección de inicio y fin de rango en los campos correspondientes

      Puertos IPv6

      Mediante esta opción podrás abrir los puertos que desees para equipos conectados a tu red en la configuración IPv6.

      o Nombre de la regla de puertos:introduce el nombre que quieras asignarle a la regla de puertos y te permita identificarla de manera intuitiva. Cada una de estas reglas ocupará una entrada en la tabla.

      o Dirección IPv6 global del dispositivo:Indicar la dirección IP v6 global del dispositivo al cual se quiere aplicar la regla de puertos definida.

      o Protocolo: selecciona el protocolo al que se aplica la regla: TCP, UDP o ambos

      o Abrir el puerto o rango de puertos:indicar si la regla está definida por un puerto o por un rango de estos. Para abrir un rango de puertos usar el formato 5001: 5010

      Pulsa el botón "Añadir". Una vez pulsado se debe crear una entrada en la tabla con la regla de puertos definida

      Si deseas deshabilitar temporalmente la regla de puertos, pulsa el botón "Desactivar". Cuando quieras volver a habilitarla pulsa el botón "Activar".

      Para eliminar una entrada de la tabla de manera definitiva pulsa el botón "Borrar". También tienes la opción de eliminar todas las entradas de la tabla de manera rápida mediante el botón "Borrar todos"

  • Cambio contraseña Router
      • Cambio contraseña Router:

      Mediante esta opción podrás cambiar la contraseña de acceso a tu router. Por defecto la contraseña de acceso, es la definida en la pegatina que podrás encontrar debajo del router denominada "Datos de acceso al router". La nueva contraseña que establezcas para entrar a tu Router debe ser alfanumérica y se recomienda que sea de al menos 8 caracteres.

  • Actualizaciones Firmware

      • Esta opción permite que actualizar la versión Firmware de tu Router de dos formas diferentes:

      Actualiza el Firmware desde tu PC:

      Si dispones de la última versión Firmware y deseas actualizar tu Router de manera manual, pulsa el botón "Buscar" para seleccionar el archivo y cargarlo en tu Router.

      Actualiza el Firmware desde la red de Movistar:

      Mediante esta opción, si hay nuevas versiones Firmware con las que puedes actualizar tu Router, se te indicará mediante un mensaje indicando la última versión de FW disponible. Únicamente debes pulsar el botón "Actualizar" y tu equipo quedará actualizado a la última versión disponible en ese momento. Si al pulsar la opción "Actualiza el Firmware desde la red de Movistar" no se detecta ninguna actualización, se te informará de esto y será suficiente con que pulses el botón "Cerrar".

  • Otras funcionalidades
      • Crear perfil o recuperar uno guardado

      Esta opción te permitirá guardar en un archivo.img la configuración concreta del router, con todos los parámetros para recuperarla más adelante. Un perfil es un conjunto de reglas y parámetros que se aplican para crear una configuración determinada. Se guardan todos los parámetros que definen tu router, tales como los de la red inalámbrica, configuración de red local, filtrado MAC, apertura de puertos, etc.

      Puedes guardar este perfil con el nombre que desees y cargar el perfil cuando lo desees presionando el botón "Guardar". Al hacer click en este botón se te mostrará un resumen del perfil donde debes añadir el nombre que quieras darle al perfil. Pulsa el botón "Aceptar" para almacenarlo. Si finalmente no quieres almacenarlo haz click en el botón "Cancelar".

      Para recuperar un perfil previamente guardado haz click en el botón "Cargar" y selecciona uno de los perfiles que se muestren. Cuando selecciones uno de ellos se te mostrará un resumen con todos los parámetros. Si deseas guardar el perfil haz click en "Aceptar" y si finalmente no deseas guardarlo haz click en "Cancelar"

      Restaurar a valores a fábrica

      Esta opción te permite restaurar la configuración por defecto de fábrica del Router. Debes tener en cuenta que si empleas está opción, se borrarán todos los parámetros configurados en tu Router, como por ejemplo la contraseña de acceso al Router, nombre y contraseña de tu red inalámbrica, etc.tomando de nuevo los valores de la configuración inicial.

      Firewall

      Podrás configurar el Firewall que trae el router para proteger nuestro PC de ataques externos. Te recomendamos no desactivarlo para tener protegido tu PC. Si deseas configurarlo solo tienes que hacer click aquí, o bien a través de la opción: Otras funcionalidades-->Firewall

      Protocolo UPnP

      Esta opción te permitirá activar/desactivar la funcionalidad conocida como Universal Plug and Play (UPnP).

      Permite descubrir de manera automática un dispositivo nuevo conectado en tu red local y puedan ser utilizados casi de manera instantánea sin ser configurados. Una vez que el nuevo equipo se haya configurado correctamente, puedes desactivarlo. Para más información consulta el manual del fabricante del router.

  • Ayuda
      • Ayuda configurador web:

      Tutorial sobre cada uno de los apartados del configurador web.

      Acerca de:

      Resumen con información relevante acerca de la configuración de tu Router y tu conexión.

      Este apartado muestra los siguientes parámetros:

      Información del equipo:

      Permite identificar tu router y las diferentes versiones de Hardware, Software y Firmware, etc. También podrás identificar de una manera rápida si tu equipo está conectado o no a Internet.

      Información WAN:

      Dirección IP WAN (Internet):Dirección IP pública de tu router.

      Información Red Local (LAN):

      Máscara de subred: muestra el valor de tu máscara de subred.

      DHCP: en caso de estar activado el servidor DHCP mostrará las direcciones de inicio y final de rango de direcciones que se pueden asignar a los equipos asociados a tu Router.

      Servidor DNS:muestra el valor de tu servidor DNS.

      Para más información sobre estos parámetros de tu red local consulta el apartado "Red Local" de este tutorial.

      Información Red inalámbrica

      Estado WiFi: muestra si estás o no conectado de manera inalámbrica.

      Nombre WiFi: Nombre de tu red inalámbrica.

      Oculto Nombre WiFi:Te indica si tienes o no oculto el nombre que identifica tu red inalámbrica.

      Tipo cifrado: muestra el cifrado que has establecido a tu red inalámbrica.

      Encriptación: muestra la encriptación establecida a tu red inalámbrica

      Para más información sobre estos parámetros de tu red inalámbrica consulta el apartado "WiFi" de este tutorial

      Información Movistar TV

      Servicio contratado:muestra si el cliente tiene un servicio Trío o un Dúo contratado

      Tipo de direccionamiento:Indica el tipo de direccionamiento de la línea del cliente. El direccionamiento dinámico es el más habitual. Para tener direccionamiento estático, el cual deberás contratar con Movistar y configurar los valores que te han proporcionado.

      Servicio de voz:Indica si tienes servicio de Voz sobre IP

      Número de Teléfono:muestra tu número de teléfono

      Información de acceso

      Id_ONT: número de acceso a tu ONT.
